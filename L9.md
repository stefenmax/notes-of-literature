+ 计算复杂度降低  
一开始的计算复杂度为O(N^4),iCT-Net减到了O(N^2 x Nc) N代表医学图像的大小 Nc代表投影数。  
此文提出的DNA框架将计算复杂度降到了O(C x N x Nc) C 是一个可调整的超参数 其远远小于N
